# Font Attribution

This project bundles two UI fonts for Android Flutter to ensure consistent
rendering across devices:

- `DM Sans` (sans text layer)
  - Source: https://github.com/google/fonts/tree/main/ofl/dmsans
  - License: SIL Open Font License 1.1
  - License file: `docs/licenses/fonts/DM_SANS_OFL.txt`
- `IBM Plex Mono` (mono metrics/meta layer)
  - Source: https://github.com/google/fonts/tree/main/ofl/ibmplexmono
  - License: SIL Open Font License 1.1
  - License file: `docs/licenses/fonts/IBM_PLEX_MONO_OFL.txt`

Bundled font assets are located at:

- `apps/android_flutter/assets/fonts/dm_sans/`
- `apps/android_flutter/assets/fonts/ibm_plex_mono/`
